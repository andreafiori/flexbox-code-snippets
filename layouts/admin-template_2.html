Flexbox Based Admin Dashboard v. 0.1

This is View Template

I originally created this for use in an ASP.NET MVC 5 / Core 1.0 project. You would put the @RenderBody() here (inside the <div class="page">) if this were a real _Layout.cshtml page.

Features

  • CSS3 Flexbox! (That was the easiest part!)
  • Only HTML5 and CSS3 ... NO JAVASCRIPT! NO BOOTSTRAP!
  • NO IMAGES ... created an SVG Icon System using Iconic Open
  • Used the "Checkbox Hack" for Sidebar and Footer
  • I only use Flat UI Colors

Planned Improvements

  • More interactive elements in the header and footer areas (logout, view profile, messages, etc. typical admin dashboard stuff)
  • Resizeable?
  • Responsive / Mobile Friendly
  • Sass
  • Fix tiny bugs here and there
